# Bare_Metal

The course I'm following is:<br>
ARM Cortex-M7 STM32F7 Bare-Metal Programming From Ground Up™, By: Israel Gbati<br><br>
Link to store page: https://www.udemy.com/course/arm-cortex-m7-stm32f7-bare-metal-programming-from-ground-uptm/?couponCode=KEEPLEARNING<br>
