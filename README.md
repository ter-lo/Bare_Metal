# Bare_Metal
This is my journey learning Bare Metal Systems using the STM family of Microcontrollers

The course I'm following is:
ARM Cortex-M7 STM32F7 Bare-Metal Programming From Ground Up™
By: Israel Gbati
Link to store page: https://www.udemy.com/course/arm-cortex-m7-stm32f7-bare-metal-programming-from-ground-uptm/?couponCode=KEEPLEARNING

In his course Israel Gbati uses the STMicroelectronics STM32F7-NUCLEO which has an ARM Cortex-M7 microcontroller.
I have ordered this board but have not received it yet, I'm enjoying not having the exact documentation because I'm working with a different board and microcontroller entirely, documented below.

My device: STMicroelectronics STMF030R8-NUCLEO which has an ARM Cortex-M0 microcontroller.

My documentation differs from the course because I'm using a different board and mcu, so throughout this document you'll see my thought process when searching for components and how to use them.

I will provide links to the documentation as I find publicly available sources (Most will be from the ST website).
